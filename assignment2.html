Attendance Management App

Attendance Management App

Built with: MIT App Inventor + Google Forms + Google Sheets

About MIT App Inventor

MIT App Inventor is a web-based platform that allows anyone to create Android apps using a simple drag-and-drop interface. It is designed for people with little or no programming experience and is widely used for educational and prototyping purposes. With MIT App Inventor, developers can quickly design the app UI and use visual blocks to control logic, making app development fast, fun, and beginner-friendly.

Features of the Attendance App

  • Simple and intuitive user interface
  • No coding required
  • Collects attendance using Google Forms
  • Stores data in real-time in Google Sheets
  • Easy to update and customize
  • Works on all Android devices

How the App Works

  1. User opens the app and taps "Open Attendance Form"
  2. The form opens inside the app via WebViewer
  3. User enters name, date, and selects status (Present/Absent/Late)
  4. Data is automatically submitted to a linked Google Sheet
  5. Google Sheets records the timestamp and attendance details

This process is seamless, fast, and fully automated without writing a single line of code!
